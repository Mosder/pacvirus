# A virus that installs pacman on your system

A simple virus made for a uni class.
<br>
Installs pacman package manager on your system and makes a seperate root directory for it which is added to the PATH variable. It also adds aliases for some apt commands, so pacman is used instead.
<br><br>
<sub><sup>I think it goes without saying, but don't run it outside of a VM or a burner PC.</sup></sub>
